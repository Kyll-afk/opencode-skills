# opencode-skills

Ten cognitive execution modes for OpenCode, built on the Higher Mind philosophy.

## Available Skills

| Skill | Description |
|-------|-------------|
| `/hm-init` | Start a new project with the best tools and structure |
| `/hm-architect` | Design system architecture, patterns, and data models |
| `/hm-engineer` | Validate code across all layers |
| `/hm-designer` | Validate interface against the highest standard |
| `/hm-qa` | Test everything, find gaps, verify it works |
| `/hm-deploy` | Validate infrastructure and deploy security |
| `/hm-security` | Security audit and vulnerability detection |
| `/hm-refactor` | Improve code structure and reduce technical debt |
| `/hm-document` | Create ADRs, README, API docs |
| `/hm-optimize` | Profile and optimize performance |

Each skill has its own SKILL.md in its subfolder.
