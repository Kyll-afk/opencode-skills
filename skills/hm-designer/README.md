# hm-designer

Validate interface against the highest software design standard. Sophistication, delight, and beauty.

## Usage

In OpenCode:
```
/hm-designer
```

## When to Use

- UI/UX validation
- Design review
- Ensuring visual consistency
- Checking accessibility
- Validating micro-interactions

## Reference

Products like Apple, Stripe, Linear, Airbnb, and Vercel set the standard.
