# hm-security-auth

Authentication security checklist. Best practices for secure auth implementation.

## Usage

In OpenCode:
```
/hm-security-auth
```

## When to Use

- Authentication implementation
- Password security
- Token management
- Session security
- Auth flow validation

## Focus Areas

- Password hashing (bcrypt/argon2)
- Token storage (httpOnly cookies)
- JWT security
- Session management
